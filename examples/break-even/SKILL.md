# Break-Even Calculator（保本点计算）

## When to Use

当用户请求涉及以下内容时使用此技能：

- 计算保本点 / 盈亏平衡点
- 分析达到盈利所需的最低销量
- 评估价格或成本变动对保本点的影响
- 安全边际分析

**触发关键词：** 保本点、盈亏平衡、break-even、保本分析

## Steps

1. **提取参数** — 从用户输入中识别以下关键参数：
   - 固定成本（Fixed Costs, FC）
   - 单位变动成本（Variable Cost per Unit, VC）
   - 单位售价（Selling Price per Unit, P）
   - 当前销量或预期销量（可选，用于安全边际计算）

2. **参数校验** — 确认：
   - P > VC（否则不存在保本点）
   - FC ≥ 0
   - 所有数值为非负数

3. **计算保本销量** — 公式：
   ```
   保本销量 = FC / (P - VC)
   ```

4. **计算保本销售额** — 公式：
   ```
   保本销售额 = 保本销量 × P
   ```

5. **计算安全边际**（如提供了当前销量）：
   ```
   安全边际率 = (当前销量 - 保本销量) / 当前销量 × 100%
   ```

6. **格式化输出** — 按配方定义的 output_format 生成结构化报告。

## Pitfalls

- ⚠️ **多产品场景**：此技能仅适用于单一产品。多产品需使用加权平均边际贡献率。
- ⚠️ **阶梯成本**：固定成本可能随产量阶梯式变化，线性假设可能不成立。
- ⚠️ **价格折扣**：大批量折扣会改变单位售价，需分段计算。
- ⚠️ **缺失参数**：如果用户未提供必要参数，必须明确询问，不得假设。
