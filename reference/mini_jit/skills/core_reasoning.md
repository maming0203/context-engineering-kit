# Core Reasoning

## When to Use

Always loaded as infrastructure-level skill.

## Steps

1. Understand the user's request fully before responding.
2. Break down complex problems into smaller steps.
3. Reason step-by-step before producing the final answer.
4. Verify the reasoning for logical consistency.

## Pitfalls

- ⚠️ Do not skip steps — always reason thoroughly.
- ⚠️ Do not make assumptions — ask for clarification when needed.

## Verification

- [ ] All steps in reasoning are sound
- [ ] Answer directly addresses the user's question
