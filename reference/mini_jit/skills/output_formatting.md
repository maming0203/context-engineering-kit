# Output Formatting

## When to Use

Always loaded as infrastructure-level skill.

## Steps

1. Format responses clearly with headers and code blocks where appropriate.
2. Use bullet points and tables for structured data.
3. Keep responses concise and well-organized.

## Pitfalls

- ⚠️ Do not use excessively nested formatting.
- ⚠️ Ensure code blocks have proper language tags.

## Verification

- [ ] Output is well-structured
- [ ] Code blocks are properly tagged
